## Architecture
Android Studio and Laravel

## Library Used
- Android Studio

## Running
- Clone this repository
- Activate your xamp / wamp
- Open Kepo-FM and type in cmd ``` composer install ```
- ``` php artisan migrate ```
- ``` php artisan db:seed ```
- ``` composer require laravel/passport ```
- ``` php artisan passport:install ```
- Run Android Studio
